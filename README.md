## 📌 About This Project

This project was built as part of the **Frontend Developer Internship assignment** for **Celebrare** — a platform that lets users create beautiful digital invitations, greeting cards, and celebrations.

The assignment involved building a **text-on-image editor** — a core feature of Celebrare's product — where users can add, style, drag, and customize text overlays on images in real time.

---

## ✨ Features

- 🖼️ **Canvas-based text editor** — Add text directly on top of images
- ✍️ **Font customization** — Change font family, size, weight, and style
- 🎨 **Color picker** — Real-time text color customization
- 📐 **Drag & reposition** — Move text freely across the canvas
- 🔛 **Alignment controls** — Left, center, and right text alignment
- ➕ **Multiple text layers** — Add and manage multiple text elements
- 🗑️ **Delete text** — Remove individual text layers
- 📱 **Responsive UI** — Works across screen sizes

## 🚀 Getting Started

### Prerequisites

Make sure you have **Node.js ≥ 18** and **npm** installed.

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/KUNDANIOS/celebrare.git

# 2. Navigate into the project
cd celebrare

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
